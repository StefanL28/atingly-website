# Session — 11 mars 2026

## Ce qu'on a fait

Implémenté la landing page marketing complète de **Swoop** — une plateforme de shopping discount par chat, disponible à Abidjan, Côte d'Ivoire.

---

## Stack technique

| Couche | Choix |
|---|---|
| Framework | Astro 5 (output statique) |
| Styling | Tailwind CSS v4 |
| Animations | GSAP + ScrollTrigger |
| Police | Plus Jakarta Sans (Google Fonts) |
| Déploiement cible | Hostinger `public_html` |

---

## Structure du projet

```
src/
├── layouts/Layout.astro        # Base HTML, meta, Google Fonts
├── pages/index.astro           # Page unique composant tout
├── components/
│   ├── Navbar.astro            # Nav sticky + menu mobile plein écran
│   ├── Hero.astro              # Hero 2 colonnes avec mockup téléphone CSS
│   ├── SocialProof.astro       # Barre de stats (count-up animé)
│   ├── FeatureSection.astro    # Sections features alternées (réutilisable)
│   ├── BentoGrid.astro         # Grille de 6 cards features secondaires
│   ├── BentoCard.astro         # Card individuelle bento
│   ├── HowItWorks.astro        # 3 étapes avec ligne de connexion
│   ├── Testimonials.astro      # 3 cards témoignages (scroll horizontal mobile)
│   ├── TestimonialCard.astro   # Card témoignage individuelle
│   ├── AvatarStack.astro       # Stack d'avatars superposés
│   ├── FAQ.astro               # Accordéon FAQ (vanilla JS)
│   ├── FAQItem.astro           # Item FAQ individuel
│   ├── FinalCTA.astro          # Section CTA finale sombre + boutons app stores
│   ├── Footer.astro            # Footer 3 colonnes
│   └── ui/
│       ├── Button.astro        # Composant bouton réutilisable (pill, variants)
│       └── SectionHeading.astro # En-tête de section réutilisable
├── scripts/animations.ts       # Toutes les animations GSAP ScrollTrigger
├── styles/global.css           # Tailwind + CSS custom properties (palette)
└── data/
    ├── features.ts             # Données features principales + bento
    ├── testimonials.ts         # 3 témoignages
    └── faq.ts                  # 6 questions/réponses
```

---

## Sections de la page (de haut en bas)

1. **Navbar** — Logo + liens + CTA pill ; transparent → blanc + ombre au scroll ; hamburger mobile avec overlay plein écran
2. **Hero** — Badge "🇨🇮 Disponible à Abidjan", headline avec mots accentués en orange, sous-titre, CTA, stack d'avatars, mockup iPhone CSS avec interface chat, carte flottante "-50%", badge étoile flottant
3. **Social Proof** — "10 000+ Utilisateurs | 500+ Offres | 50+ Partenaires" avec animation count-up au scroll
4. **Feature Sections** — 3 blocs alternés téléphone+texte avec blobs décoratifs SVG (chat, deals personnalisés, carte locale)
5. **Bento Grid** — 6 cards : Notifications, Interface simple, Gratuit, Partenaires vérifiés, Offres quotidiennes, Sécurisé
6. **How It Works** — 3 étapes numérotées avec ligne de connexion desktop
7. **Testimonials** — 3 cards avec étoiles, auteur + quartier d'Abidjan ; scroll horizontal sur mobile
8. **FAQ** — 6 questions avec accordéon JS (expand/collapse smooth, icône +/×)
9. **Final CTA** — Fond sombre, headline, boutons App Store + Google Play
10. **Footer** — Logo, tagline, liens rapides, contact, réseaux sociaux, copyright
